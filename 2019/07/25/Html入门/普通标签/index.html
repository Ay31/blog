发表于 2019-07-25 更新于 2021-04-14 分类于 Html入门
各种标签

第一部分

  1. <body> 网页上显示的内容
  2. <p> 段落
  3. <hx>1-6 级标题
  4. <strong>强调 粗体
  5. <em>强调 斜体
  6. <span>为文字设置单独样式
  7. <q>短文本引用 会自动加双引号 语义引用他人的话
  8. <blockquote>长文本引用 会左右自动缩进
  9. <br> 换行
  10. &nbsp; 空格 注意分号
  11. <hr>水平横线
  12. <address>地址 不能包含在<p>内
  13. <code> 代码 (一行 少量）
  14. <pre> 代码 (多行)

第二部分

  1. <ul><li><li><ul> 无序标签

  2. <ol><li><li><ol> 有序标签

  3. <div> 容器 划分出独立逻辑部分\

  4. 表格标签

    1. <tr> 表格的一行 有几个 tr 表格就有几行
    2. <th> 表格的头部的一个单元格 表格表头
    3. <td> 表格的一个单元格 一行中有几个 td 就有几列

    如果不加<thead><tbody><tfooter> , table 表格加载完后才显示。加上这些表格结构， tbody 包含行的内容下载完优先显示，不必等待表格结束后在显示，同时如果表格很长，用 tbody 分段，可以部分一部分地显示。（通俗理解 table 可以按结构一块块的显示，不在等整个表格加载完后显示。）

  5. <caption>表格标题 加在<table></table>开始

  6. <table summary=""> 为表格添加摘要

第三部分

  1. <a href="目标网址" title="鼠标滑过显示的文本">链接显示的文本</a>
    a. target="\_blank" 在新窗口打开
    b. href="mailto:xxx@xx.com" 如果 mailto 后面同时有多个参数的话，第一个参数必须以“?”开头，后面的参数每一个都以“&”分隔。
  2. <img src="图片地址" alt="下载失败时的替换文本" title = "提示文本">
