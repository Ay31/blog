表单标签

发表于 2019-07-25 更新于 2021-04-14 分类于 Html入门
嵌套规则和某些特殊标签
  1. <form method="传送方式" action="服务器文件"> </form>
  2. <input type="text/password" name="名称" value="文本"/>
  3. <textarea>这里可添加默认文本</textarea> 文本域
  4. <input type="radio/checkbox" value="值" name="名称" checked="checked"/>
    radio 是单选框 checkbox 是复选框 当 checked 为 checked 时 该选项默认选中
    同一组的单选按钮，name 取值一定要一致，否则无法起到单选作用
  5. <select>
    <option value="向服务器提交的值" selected="selected">浏览器显示的值</option>
    <option></option>
</select>
    下拉框 selected 为默认选中的值
  6. <select multiple="multiple"> 下拉列表也可以进行多选操作，在<select>标签中设置 multiple=”multiple”属性，就可以实现多选功能，在 windows 操作系下，进行多选时按下 Ctrl 键同时进行单击
  7. <input type="submit" value="提交"> 只有 type 为 submit 时按钮才具有交作用
  8. <input type="reset" value="重置"> 只有当 type 值设置为 reset 时，按钮有重置作用
  9. <label for="控件名称"> for 属性中的值一定要与相关控件的 id 值相同 ，如你在 label 标签内点击文本，就会触发此控件。
    如单选框中 ·男，若没有 label 标签，用户只能点击到圆点才能选中，但使用 label 标签，点击到文本”男”也可选中。
