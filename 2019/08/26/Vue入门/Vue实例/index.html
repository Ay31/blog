Vue实例

创建一个Vue实例

一个Vue应用应该由通过new Vue创建的跟Vue实例以及可选的嵌套的、可复用的组件树组成。

所有的 Vue 组件都是 Vue 实例，并且接受相同的选项对象（某些特有选项除外）。

数据与方法

Vue实例中，data对象的所有属性加入到响应式系统中，只有data对象中的对象才是响应式的，除了数据属性，还有’$’前缀的实例属性和方法。

实例生命周期钩子

Vue实例创建过程中会运行不同的生命周期函数（created，mounted，updated，destroyed）,不要使用箭头函数（没有this）。
